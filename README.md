# かいもの 🧻

夫婦で共有する消耗品リスト。なくなりそうな日用品をメモして、買ったらタップで消すだけのミニマルなPWA。

- **アプリ本体**: [`docs/index.html`](docs/index.html)（単一HTML。フレームワーク・ビルドなし）
- **同期**: Firebase Realtime Database。URLフラグメント（`#l=...`）の推測不能なリストIDが実質の鍵で、IDはこのリポジトリには含まれない
- **配信**: GitHub Pages（Settings → Pages → Branch: `main`, Folder: `/docs`）
- **オフライン対応**: `docs/sw.js`（ネットワーク優先・失敗時キャッシュ）

## 使い方

| 場面 | 操作 |
|---|---|
| 消耗品が切れそう | 入力して追加（過去に買ったものは「ストック」をタップするだけ） |
| 店でカゴに入れた | 行をタップ（購入済み→ストックへ） |
| 誤タップ | 4秒以内に「元に戻す」 |
| もう買わないもの | 長押しで完全削除 |
